# inmetrss

<!-- badges: start -->
<!-- badges: end -->

Fetch and parse INMET alerts.

## Installation

You can install the development version of inmetrss from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("rfsaldanha/inmet_alerts")
```

## Example
